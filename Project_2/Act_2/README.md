# Libreria Nexus Books

![Nexus_books_logo](./public/Nexus_Librery.svg)

## Objetivo del proyecto

El propósito de Librería Nexus es crear una plataforma moderna que promueva la lectura y conecte a las personas a través de la cultura, la tecnología y el diseño.
El proyecto también forma parte del máster en Desarrollo Front-End y UX/UI, integrando prácticas de usabilidad, accesibilidad y diseño responsive.

## Características principales

- Landing Page interactiva con secciones dinámicas.

- Catálogo de libros con información detallada.

- Búsqueda por título o autor.

- Carrito de compras con gestión de productos.

- Sección "About Us" para conocer la historia del proyecto.

- Diseño responsive optimizado para móviles, tabletas y escritorio.

## Tecnologías utilizadas

- React

- Vite

- TailwindCSS

- React Router DOM

- Lucide React
 (iconos SVG)

- Vercel
 (despliegue)

## Instalación y ejecución

 Clona el repositorio:

```bash

 git clone https://github.com/tuusuario/libreria-nexus.git
```

Instala las dependencias:

```bash
npm install
```

Ejecuta el servidor de desarrollo:

```bash
npm run dev
```
