# App Libreria Nexus Books

![Nexus_books_logo](./assets/Nexus_Books_logo_mobile.svg)

## Objetivo del proyecto

El objetivo de este proyecto es desarrollar una aplicación móvil multiplataforma para una librería universitaria y espacio multifuncional, que integra la venta de libros, una zona de co-working y una cafetería, utilizando React Native y Expo.

La aplicación busca replicar la funcionalidad de una versión web previamente desarrollada en React, adaptándola a una experiencia nativa para dispositivos Android e iOS, enfocándose principalmente en el diseño de la interfaz de usuario y la experiencia de navegación.

Además, el proyecto tiene como finalidad:

- Consumir datos desde una API simulada (Back-End simulado).
- Implementar navegación mediante Stack y Tabs.
- Aplicar estilos con NativeWind.
- Incorporar retroalimentación háptica para mejorar la experiencia del usuario.
- Garantizar su funcionamiento en dispositivos virtuales y físicos.

Este proyecto forma parte de una actividad académica del máster y demuestra el uso de tecnologías modernas para el desarrollo de aplicaciones móviles multiplataforma.

## Características principales

- Aplicación móvil multiplataforma desarrollada con React Native y Expo.

- Consumo de datos desde una API simulada (Back-End simulado).

- Diseño de interfaz moderna utilizando NativeWind para los estilos.

- Navegación implementada con Stack y Tabs.

- Experiencia de usuario mejorada con retroalimentación háptica.

- Implementación de una pantalla de inicio (Landing) y múltiples vistas adicionales.

- Uso de fuentes personalizadas para mejorar la estética de la aplicación.

- Adaptación y reutilización de lógica de una versión web creada previamente con React.

- Compatible con Android e iOS.

- Pruebas realizadas en dispositivos físicos y emuladores.

## Tecnologías utilizadas

## Instalación y ejecución

 Clona el repositorio:

```bash

 git clone 
```

Instala las dependencias:

```bash
npm install
```

Ejecuta el servidor de desarrollo:

```bash
npm start
```
