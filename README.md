# Aplicaciones Moviles Multiplataforma

![Universidad_Internacional_Unir_logo](./img/Logo_UNIR.svg)

## Descripción

Este repositorio contiene los proyectos desarrollados durante la asignatura Aplicaciones Móviles Multiplataforma del Máster en Desarrollo Front-End. El objetivo principal de esta materia es diseñar y desarrollar aplicaciones web y móviles multiplataforma utilizando React y React Native, aplicando buenas prácticas de arquitectura, usabilidad y optimización de rendimiento.

## Objetivos de la materia

- Comprender las diferencias y similitudes entre React y React Native.

- Diseñar interfaces modernas y funcionales para entornos web y móvil.

- Implementar componentes reutilizables y estructurar correctamente una aplicación.

- Conectar las aplicaciones con APIs externas para mostrar y gestionar datos.

- Publicar y probar aplicaciones móviles mediante Expo.

## Tecnologías utilizadas

- React.js → para el desarrollo de aplicaciones web.

- React Native → para el desarrollo de aplicaciones móviles nativas (Android / iOS).

- JavaScript (ES6+) → lenguaje principal de programación.

- Node.js y npm → para la gestión de dependencias y ejecución del entorno.

- Expo → entorno de desarrollo y despliegue de apps móviles.

- CSS / Styled Components → para la maquetación y diseño de interfaces.

- Git / GitHub → para control de versiones y gestión del código fuente.
